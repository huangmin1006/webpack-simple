# webpack-simple

#### 基于最新 **webpack: 4.34.0**


1. 初始化一个项目

    ```
    ├── dist                      打包输出目录，只需部署这个目录到生产环境
    ├── package.json              项目配置信息
    ├── node_modules              npm 安装的依赖包都在这里面
    ├── src                       我们的源代码
    │   ├── components            可以复用的模块放在这里面
    │   ├── index.html            入口 html
    │   ├── index.js              入口 js
    │   ├── shared                公共函数库
    │   └── views                 页面放这里
    └── webpack.config.js         webpack 配置文件
    ```
